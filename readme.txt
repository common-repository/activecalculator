== Installation ==
= Automatic installation =
1. Go to the Plugin menu and click on the **Add New** Button
2. Search for **ActiveCalculator**
3. Install and activate the ActiveCalculator plugin from there

= Download & Upload: =
1. Download the **ActiveCalculator** plugin (*.zip file*)
2. In your admin area, go to the Plugins menu and click on **Add New**
3. Click on **Upload Plugin** and choose the **`activecalculator.zip`** file and click on **Install Now**
4. Activate the plugin and Enjoy!

= Manually: =
1. Download and upload the **activecalculator** plugin to the **`/wp-content/plugins/`** directory
2. Activate the plugin through the Plugins menu in WordPress

= Use Block: =
1. Go to the WordPress Block/Gutenberg Editor
2. Add one or more blocks from the block category called **ActiveCalculator** in the Gutenberg editor
3. Make sure you have a calculator ready to be embedded and Enter the ID of the calculator you want to embed in the **ActiveCalculator** block.
4. Enjoy!
